# Phase 2 Backlog Seeds

1. **Pause & accessibility menu**
   - Overlay toggled by host; includes sound toggle, restart (if allowed), and control recap.
2. **Daily challenges**
   - Playlist generator keyed by current date with themed modifiers and shared seed.
3. **Seedable runs UI**
   - Allow players to input a seed for reproducible playlists; expose shareable codes.
4. **Speedrun mode**
   - No pauses, tighter timers, leaderboard integration via telemetry sink.
5. **Theme packs**
   - Color + audio palettes, switchable per session; host broadcasts theme to applets via `host.setTheme`.
6. **Adaptive difficulty**
   - Adjust stage weights based on recent success streaks and completion times.
7. **Applet sandbox health monitor**
   - Detect long-running tasks, memory leaks, and unhandled promise rejections.
8. **In-app tutorial / attract mode**
   - Host-controlled preview reel with scripted inputs for each applet.
